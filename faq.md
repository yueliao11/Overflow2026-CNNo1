# FAQ / 常见问题

## 关于 Sui Overflow 2026

### 什么是 Sui Overflow 2026？

Sui Overflow 2026 是 Sui 的旗舰级全球线上黑客松，聚焦有意义的产品、真实世界的应用和长期的生态增长。无论你是在构建 AI 原生应用、DeFi 产品还是全新的链上体验，Overflow 都旨在帮助建设者从 Idea → Product → 生态影响。

### 为什么要参加？

- 与 Sui Foundation 深度合作
- 连接全球顶级开发者和投资人
- 争夺超过 **$250,000+** 的奖金池
- 获胜团队有机会获得加速器引荐、融资机会和 Office Hours 辅导
- 黑客松不只是比赛——优秀项目可进入 **Hackathon → Shortlisting → Demo Day → Winning → Mainnet Launch** 的完整路径

### 谁可以参加？

**任何人都可以参加！** 无论你是资深 Web3 开发者、设计师还是刚刚接触区块链的新人，都没有经验要求——带上你的想法和动力就行。

### 需要线下参加吗？

**不需要。** Sui Overflow 是纯线上黑客松，注册、开发和评审都在线上完成。会有一些线下的开发者聚会和 Workshop 活动，但这些都是可选的，不影响评审。

### 如何组队？

注册时可以标记你在寻找队友，会被加入组队目录。你也可以在官方 Sui Overflow Discord 频道中寻找队友。我们还会组织一些组队活动，具体时间在 Discord 中公布。

### 什么时候开始开发？

项目应从官方黑客松启动后开始构建，以确保公平性。

### 如何选择赛道？

选择最能代表你项目的赛道。评委将根据所选赛道进行评估，所以选择与你想法最契合的那个。

### 有问题可以联系谁？

官方联系邮箱：devrel@sui.io

每个项目选择一个赛道提交。选择最匹配你项目的那个赛道。

### 什么是 DeepBook Predict？

DeepBook Predict 是 Sui 上的可编程、波动率曲面定价预测协议。已在测试网上线，支持滚动亚小时 BTC 预言机，使用 `dUSDC` 作为报价资产。详见 [DeepBook 赛道说明](tracks/deepbook.md)。

### Walrus 赛道一定要用 MemWal 吗？

不一定要用 MemWal。Walrus 赛道鼓励使用 Walrus 作为可验证数据平台构建 AI Agent 或工作流，可以直接使用 Walrus 的持久存储，也可以通过 MemWal（Walrus Memory）SDK 集成。详见 [Walrus 赛道说明](tracks/walrus.md)。

## 关于奖金

### 奖金怎么发放？

采用 **50/50 分期发放** 模式：
- 获奖公布时发放 **50%**
- 成功部署主网后发放剩余 **50%**
- 如果截止时已部署主网 → **100% 一次性发放**

主网部署需要满足 Sui 团队和/或赛道赞助方定义的最低功能要求。

### 有学生奖吗？

有！**10 个名额，每人 $2,500 USD**。团队至少 **50%** 成员需要是在校学生。

### 部署主网有什么要求？

主网部署需要满足 Sui 团队和/或赛道赞助方定义的最低功能要求。项目需要是功能完整的产品，而非纯技术演示。

## 关于提交

### 需要提交什么材料？

| 字段 | 要求 |
|------|------|
| 项目名称 | 简洁明了 |
| 项目描述 | 做什么，为什么重要 |
| 项目 Logo | 1:1 比例 (JPG/PNG) |
| GitHub 仓库 | 评审期间必须公开 |
| Demo 视频 | 必须，YouTube 优先，≤ 5 分钟 |
| 网站 | 可选，强烈建议 |
| 部署环境 | Testnet 或 Mainnet |
| Package ID | 如已部署链上 |

### 项目还没写完可以提交吗？

可以。先提交已有内容，后续可以继续迭代更新。但评委评审时会看项目的完整度和可用性。

## 关于中文区

### 这个仓库是做什么的？

本仓库是 **Sui Overflow 2026 中文区的项目登记入口**，用于：
- 统计中文区参赛项目
- 抽奖（Sui Play 游戏机）
- 发放周边

> **注意：正式参赛仍需在 [Sui Overflow 官网](https://overflow.sui.io/) 完成 DeepSurge 注册和项目提交。**

### 怎么参与中文区抽奖？

提交一个**满足参赛标准**的项目 PR 到本仓库：
- PR 合并后自动获得 **1 次抽奖机会** + **1 份周边**
- 奖品：**Sui Play 游戏机 x1**
- 抽奖时间另行通知

### 我的项目还没写完，可以先登记吗？

可以。只要在 PR 里填上项目名称、赛道和简介就行。GitHub 仓库链接可以先填上，后续通过新的 PR 更新。

### 一个人可以提交多个项目吗？

一个目录对应一个项目。如果想提交多个项目，可以在同一个 README 里追加，或者联系管理员。

### 团队项目怎么算？

由一名成员提交，在 README 的 Team 里列出所有成员的 GitHub ID。所有团队成员都有抽奖资格和周边。

### 不在中文区可以提交吗？

本仓库面向中文社区。如果你在中文社区活跃（微信群、Discord 中文频道等），欢迎提交。

### 周边怎么领取？

在模板里勾选"我希望接收周边"，然后通过[周边收件表单](#)单独提交收件信息（地址、联系方式等），避免在公开 PR 中暴露隐私。

## 关于技术

### 需要会 Move 语言吗？

Sui 的智能合约使用 Move 语言。如果你只是构建前端或集成工具，不一定需要写 Move。但如果涉及链上逻辑，Move 是必须的。参考 [Sui 官方文档](https://docs.sui.io)。

### 什么是 DeepSurge？

DeepSurge 是 Sui Overflow 的项目展示平台。注册参赛需要在 DeepSurge 上创建项目展示页。详见 [overflow.sui.io](https://overflow.sui.io/)。

### 有技术支持吗？

有。黑客松期间会有 Workshop 系列覆盖构建所需资源，还可以在官方 Discord 中与导师和 Overflow 团队交流。
